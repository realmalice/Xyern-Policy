Privacy Policy — Xyern

Effective Date: May 2026

Xyern is a Discord moderation, security, automod, and verification bot designed to help communities manage and protect their servers safely.

Information We Collect

Xyern may temporarily process or store limited Discord-related data required for bot functionality, including:

Discord User IDs

Server IDs

Channel IDs

Role IDs

Moderation settings

Verification settings

Logging configurations

Whitelist configurations


Xyern does not collect passwords, payment information, private files, or personal sensitive information outside of Discord-provided data.

How Data Is Used

Collected data is used only for:

Moderation features

Automod systems

Verification systems

Antiraid and antinuke protection

Logging and configuration storage

Command execution


Data Sharing

Xyern does not:

Sell user data

Share user data with third parties

Use collected data for advertising

Use collected data for AI training

Transfer data to external data brokers


Data Storage

Data is stored securely only when necessary for bot functionality and server configuration persistence.

User Privacy

Xyern respects user privacy and only accesses information required for requested bot features and Discord server management.

Third-Party Services

Xyern operates through Discord APIs and may rely on secure hosting providers for infrastructure purposes only.

Data Removal

Server administrators may request removal of server-related configuration data by contacting the bot support team.

Changes to This Policy

This Privacy Policy may be updated periodically to reflect service or legal changes.

Contact

For privacy-related questions or requests, contact the Xyern support team through official support channels.
